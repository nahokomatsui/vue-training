# Vue.js Training

## プロジェクト作成

[Vue CLI](https://cli.vuejs.org/)の`vue create`コマンドを使用してプロジェクトを作成しています。  

### 設定
以下のオプション機能を追加しています。
- [Babel](https://babeljs.io/)
- [Vue Router](https://router.vuejs.org/)
- [ESLint](https://eslint.org/) + [Prettier](https://prettier.io/)

## 開発環境構築

### 依存パッケージインストール
```bash
npm install
```

### 開発サーバー起動
```bash
npm run serve
```

## 本番用ビルド
```bash
npm run build
```

## Lint
```bash
npm run lint
```
